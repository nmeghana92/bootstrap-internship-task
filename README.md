 Bootstrap 5 Internship Task

Project Overview
This project was developed as part of my internship assignment.  
The objective was to build a responsive multi-page website using only Bootstrap 5 components via CDN.

The website includes:
- Home Page
- About Page
- Contact Page

---

 Technologies Used
- HTML5
- Bootstrap 5 (Official CDN)
- Bootstrap Grid System
- Bootstrap Utility Classes

---

 Components Implemented
- Responsive Navbar
- Hero Section
- Feature Cards
- Grid Layout
- List Groups
- Contact Form
- Footer

All components were implemented using the official Bootstrap documentation.

---
 Responsiveness
The website is fully responsive and adapts to:
- Desktop
- Tablet
- Mobile devices

Bootstrap grid system and utility classes were used to ensure responsiveness.

---

Development Process
1. Reviewed Bootstrap official documentation.
2. Explored example components (Navbar, Cards, Forms, Grid).
3. Created structured multi-page layout.
4. Ensured consistent spacing and color scheme.
5. Tested responsiveness using browser developer tools.
6. Deployed using GitHub Pages.

---

 Challenges Faced
- Understanding the Bootstrap grid system initially.
- Managing responsive breakpoints.
- Ensuring consistent layout across all pages.

These were solved by carefully reading official Bootstrap documentation and testing different layouts.

---

Time Taken
Approximately 2 hours including:
- Learning components
- Building pages
- Testing responsiveness
- Deployment

---

 Tools Used
- Bootstrap Official Documentation
- ChatGPT (for guidance and structure clarification)
- GitHub (for version control and deployment)
